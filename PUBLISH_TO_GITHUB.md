# Publish to GitHub

Create an empty public GitHub repository named `ZSH` under the `sagarkorde` account, then run:

```bash
cd C:\Users\sagar\Desktop\ZSH
git push -u origin main
git tag v1.0-paper-submission
git push origin v1.0-paper-submission
```

After publishing:

1. verify the repository is public at `https://github.com/sagarkorde/ZSH`
2. create a GitHub release from tag `v1.0-paper-submission`
